# running

Personal race calendar and build-up plans, generated by a set of Claude Code skills and published as static HTML.

## Published plans

Live site: **https://stexyz.github.io/running/**

Individual plan pages render a timeline strip (with a dynamic "now" indicator and weeks-to-goal labels), a month grid, and a race table with links to the race source pages. HTML is fully self-contained — no external assets, no build step.

## Repo layout

- `docs/` — published HTML. GitHub Pages source. Landing page (`index.html`) lists every plan.
- `plans/` — plan source-of-truth files, YAML frontmatter + notes.
- `data/races/` — scraped race calendar data from Czech running sites (gitignored; regenerate with the `race-scrape` skill).
- `.claude/skills/` — the three skills that power the workflow:
  - `race-scrape` — pulls race listings from top4running.cz, ceskybeh.cz, svetbehu.cz and normalizes to JSON.
  - `race-plan` — interactive planner. Reads scraped data, applies build-up + taper heuristics, writes a plan file.
  - `race-render` — turns a plan into a shareable HTML page.

## Sharing

The published HTML is designed to be shared with coaches or teammates who don't use GitHub. Just send the plan URL from `docs/`.
